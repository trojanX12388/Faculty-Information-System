Course Syllabus
COURSE INFORMATION

This section contains all the course information are assigned for you to handle.

  1. Introduction to Computer Science (CS101)
    Overview of computer science fundamentals.
    Introduction to programming concepts using a specific language (e.g., Python).
    Basic problem-solving and algorithm development.
  2. Data Structures and Algorithms (CS201)
    Study of fundamental data structures (arrays, linked lists, trees, etc.).
    Analysis and design of algorithms for problem-solving.
    Implementation of algorithms in a chosen programming language.
  3. Database Management Systems (CS301)
    Principles of database design and management.
    SQL programming for data retrieval and manipulation.
    Introduction to relational database concepts.
  4. Artificial Intelligence (CS401)
    Overview of artificial intelligence principles.
    Machine learning algorithms and applications.
    Natural language processing and expert systems.
  5. Computer Networks (CS501)
    Fundamentals of computer networks and protocols.
    Network architecture and design.
    Network security concepts.
  6. Operating Systems (CS601)
    Principles of operating systems.
    Process management, memory management, and file systems.
    Introduction to system-level programming.
